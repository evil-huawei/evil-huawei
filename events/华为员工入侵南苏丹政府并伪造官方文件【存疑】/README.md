# 华为员工入侵南苏丹政府并伪造官方文件【存疑】
## 事件经过
### 2014.10.3
华为员工 Chen Nan 向出口银行行长发了一封伪造的南苏丹政府文件，旨在推迟给中兴的一批项目资金。

### 2014.10.17
苏丹外交部长指出华为的不当行为。

### 2014.10.22
华为道歉，并表示这是员工的个人行为，公司不知情。
